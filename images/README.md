# Logo Files

This directory should contain the following logo files:

## Required Logos

1. **cllt-logo.png** - CLLT Lab Logo
   - Recommended size: 200x200px or similar square format
   - Should be transparent background (PNG)
   - Will be displayed in the navbar at 40px height (scales automatically)

2. **simasiaai-logo.png** - SimasiaAI Logo
   - Recommended size: 200x60px or similar horizontal format
   - Should be white/light colored for dark footer background
   - Will be displayed in the footer at 24px height
   - Should link to https://www.simasiaai.gr/

## Adding Logos

1. Place your logo files in this `images/` directory
2. Name them exactly as specified above:
   - `cllt-logo.png`
   - `simasiaai-logo.png`
3. The website will automatically use them once uploaded

## Logo Sources

- **CLLT Logo**: Create or obtain from lab branding materials
- **SimasiaAI Logo**: Available from https://www.simasiaai.gr/

## Notes

- If logos are not found, they will gracefully hide (onerror handler)
- Logos should be optimized for web (compressed PNG or SVG)
- For best results, use transparent backgrounds where appropriate
